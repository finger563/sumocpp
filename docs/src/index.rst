SumoCPP Documentation
=====================

This is a cpp library for interfacing with SUMO's TraCI
TCP API.  It handles connecting and exposes remote methods.
It has a library which allows it to be used with `ROSMOD
<http://github.com/rosmod/rosmod-pkg.git>`_.
